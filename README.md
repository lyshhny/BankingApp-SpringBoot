# Simple Banking App

A secure banking application built using **Java**, **Spring Boot**, and **Maven**.

## Features
- User Registration and Login
- Session management
- Transaction history
- Dashboard interface
- Templates using HTML/CSS

## Technologies Used
- Java 17+
- Spring Boot
- Maven
- Spring Security (if applicable)
- H2 or MySQL Database
